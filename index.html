E-Plastic

Developed an online plastic collection system to facilitate recycling.


E-Plastic is an innovative platform developed as part of my BCA final year project. The platform serves as an eco-friendly solution that connects recycling companies, plastic collectors, and customers. The goal is to streamline the process of plastic collection and recycling, making it easier for individuals and businesses to participate in eco-friendly practices.

Through E-Plastic, customers can request plastic collection services, while collectors can efficiently manage their routes and inventory. Recycling companies benefit by having direct access to collected materials, ensuring a smooth supply chain for recycling operations. The system promotes sustainability, reduces plastic waste, and raises awareness about environmental responsibility.

Parkinson Pre-Screening Tool

Machine learning-based tool for pre-screening Parkinson's disease.


The software solution that can process an image of the handwriting of the user and it can analyse or predict the possibility of Parkinson disease. This study involved applying multiple machine learning models on the data and then using the random forest method to do the prediction.


The GUI solution first takes the image as the input and then processes the image using OpenCV library. Using that it finds out the placement of the spiral drawn and the variation in the shape. Then the difference in the actual drawn shape and the main shape is saved in csv file. After that the csv file goes for the machine learning and the ML code predicts the outcome using the random forest method


Overall, after testing we have found that the solution provides us the promising results with the prediction accuracy of 74 per cent and the precision of 90 per cent. This solution can be used in the medical facilities as the pre-screening tool and its results can be improved significantly with adding more data to the training with time.

Detecting Dyslexia

Built a system to detect dyslexia through eye movement analysis.


Using eye tracking measures and machine learning models to detect dyslexia. The objective is to design a prototype called DDUET (Dyslexia Detection Using Eye Tracking) that uses a VR headset and a small camera to record eye movements while the subject reads a passage. The recorded data is processed using OpenCV and machine learning models to come up with an ML classification model. The report provides technical details on the GUI design, video processing, and machine learning algorithm used in the study. The study aims to analyze the differences in reading patterns between dyslexic and non- dyslexic students by eliminating head and body movements.
